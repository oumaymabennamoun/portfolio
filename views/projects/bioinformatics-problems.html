While searching for biomedical programming project inspiration, I came across the Rosalind.info platform, which contains a collection of bioinformatics questions ranging from low to high difficulty.

My goals for this project are to...

1. Introduce myself to bioinformatics and its computational techniques.
2. Refresh myself on the programming languages taught in my classes.
3. Strengthen my problem-solving for the platform's more complex problems.

So far, I've only had the time to submit a couple of the simplest solutions (DNA and RNA related) to the platform, as I've had to take a pause for exams and my portfolio website development. However, I'm already super interested for the next chance I get to work on these problems!
